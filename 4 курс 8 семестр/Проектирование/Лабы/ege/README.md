# ЕГЭ Информатика: локальное тестирование (учитель + ученики)

Проект состоит из 3 программ:
- `teacher_app.py` - сервер и интерфейс учителя.
- `student_app.py` - клиент ученика.
- `csv_builder.py` - сборщик `tasks.csv` по папке картинок.

## 1. Требования

- Windows 10/11
- Python 3.10+
- Tkinter (обычно встроен в Python)

## 2. Формат варианта

Вариант - это папка, где лежат картинки заданий и `tasks.csv`.

Пример:

```text
variant_01/
  tasks.csv
  1.png
  2.png
  3.png
```

`tasks.csv` без заголовка:

```csv
1.png,42,1
2.png,101010,1
3.png,C,2
```

Столбцы:
1. путь к картинке (относительно папки варианта),
2. правильный ответ,
3. баллы (необязательно, по умолчанию `1`).

## 3. Запуск

```powershell
python teacher_app.py
python student_app.py
python csv_builder.py
```

## 4. Как пользоваться

1. В `csv_builder.py` выберите папку с картинками.
2. Введите ответы, нажимайте `Сохранить текущий ответ`.
3. `tasks.csv` создается и обновляется сразу в той же папке.
4. На учителе запустите `teacher_app.py`, нажмите `Запустить сервер`.
5. На ученике запустите `student_app.py`.
6. Ученик вводит только `Имя Фамилия` и нажимает `Подключиться` - учитель ищется автоматически.
7. Учитель загружает папку варианта и отправляет задания.
8. Ученики сдают работу, учитель получает статистику.

## 5. Блокировка интернета с учителя

В `teacher_app.py` есть кнопки:
- `Отключить интернет ученикам`
- `Включить интернет ученикам`

Команда отправляется на все подключенные ученические клиенты.
На ученике применяются временные правила Windows Firewall:
- разрешен `localsubnet` и IP учителя,
- внешний интернет блокируется.

Важно:
- ученический клиент нужно запускать **от имени администратора**, иначе firewall-правила не применятся;
- статус применения/ошибки приходит обратно учителю в лог;
- IDE/Excel работают локально, но их выход в интернет также блокируется.
